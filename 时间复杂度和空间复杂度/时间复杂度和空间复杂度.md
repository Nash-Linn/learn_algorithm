### 时间复杂度

#### 1.是什么？

      执行当前算法所'花费的时间'

```
const str = '123'
for(var i=0;i<100000;i++){

}
```

#### 2.干什么？

      在写代码的过程中，就可以大概知道代码运行的快与慢

#### 3.表示

      大O表示法《解析数论》
      O表示有很多，例举：O(1)、O(n)、O(n^2)、O(log n)...

### 空间复杂度

#### 1.是什么？

#### 2.表示法

      O(1)、O(n)、O(n^2)、O(log n)...
